Frequently asked questions

Does Store Locator support for multi-stores?

Yes, it allows you to apply for multiple stores as you need.

How many ways are there to find the nearest stores?

There are 3 ways to help customers find the stores conveniently:

  • Search by distance (within the specified radius from the current location).
  • Search by area (within a specific state/ province, country, city, zip code)
  • Filter by tags (product types, product brands,...)

For the radius, you can allow using miles or kilometers within the searching from Magento Admin. Moreover, you can set for each store with link to website, contact information, and an icon to be used on the map instead of the traditional location marker.

Is there any conflict with other extension after I install Store Locator module?

Almost No, it is developed comfortably and compatible well with other extensions. If you have any issue, feel free to contact us.
